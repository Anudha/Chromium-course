# Contributing

This repository contains course materials for **CSC/CHE 591: Inside Chromium** at NC State. Contributions, corrections, and forks are welcome.

## For instructors adapting the course

1. **Fork** the repo.
2. Rename to match your institution's course number.
3. Edit `SYLLABUS.md` to reflect your calendar, assessment weights, and institution-specific policies.
4. Check every external link — chromium.org URLs occasionally move.

## For students spotting errors

If you find a factual error, broken link, or outdated reference:

1. Open an issue with the file name and a description of the problem.
2. Or open a PR with the fix.

## For the world

PRs that add:

- New demos
- Additional homework variants
- Translations
- Accessibility improvements

…are welcome.

## What not to contribute

- Solved homework answers (this repo is course materials, not solutions).
- Links to third-party blogs, Medium posts, or LLM-generated tutorials. Primary sources only — see [`resources/primary-sources.md`](./resources/primary-sources.md).

## Style

- Markdown for all docs.
- Code fences with language hints.
- Relative links between files in this repo.
- Absolute links for external sources.
