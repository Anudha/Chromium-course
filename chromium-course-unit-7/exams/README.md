# Exams

Three summative assessments.

| Exam | When | Duration | Weight | Coverage |
|---|---|---|---|---|
| [Midterm 1](./midterm-1.md) | Thursday, Week 6, in class | 80 min | 15% | Units I–II (Foundations + Rendering Pipeline) |
| [Midterm 2](./midterm-2.md) | Thursday, Week 11, in class | 80 min | 15% | Units III–V (V8 + Security + Networking) |
| [Final](./final.md) | Exam week | 2h 50min | 20% | Comprehensive + project presentation |

## Format

All exams are **closed-laptop, closed-AI, closed-notes** (single handwritten one-sided 8.5×11 note sheet allowed on midterms; two sheets for final).

## Policy on academic integrity

Using any form of AI assistant during an exam is a violation of the NC State Student Code of Conduct and will result in course failure and referral. This is stated up front because the rest of the course is AI-permissive and the line must be unambiguous.
